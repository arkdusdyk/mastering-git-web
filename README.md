# 캡스톤 디자인1 프로젝트<br><방글라데시 창업지원 프로젝트2 : AI 화학 공식 튜터>
지도 교수 : 구명완, 홍철기


### 팀 구성 :
김연요, 최규식, 고서경, 황두연

### 내용 :
스마트폰에서 카메라로 인식한(OCR엔진을 통해 전달받은) 화학 공식을 3D모델로 결합방식/ 원리를 보여주며 3D 시뮬레이션을 통해 초/중학생들이 원리를 이해하고 교육에 흥미를 가지게 하는 앱

### 사용 기술 :
* OCR 엔진 (자체개발)을 통해 화학 공식 인식
* iOS / Android
* 3D 디스플레이 및 시뮬레이션

### 기능 계획 :
* 학년 별 커리큘럼 다운로드
* 노트에 화학공식, 결합식 작성 중 정답확인을 위해 스마트폰으로 노트(손글씨) 촬영
* OCR을 통해 화학공식, 결합식 정답 확인
* 상세 버튼 클릭을 통해 해당 원소, 분자 물질 3D 형태로 크기 비교, 결합형태 출력
* 3D 모델링을 통해 시뮬레이션
